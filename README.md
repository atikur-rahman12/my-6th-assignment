# 🛒 Premium Tools Store

## 📌 Description
Premium Tools Store is a modern React-based web application where users can explore digital tools, add them to a cart, and manage their selections easily. The project focuses on clean UI, smooth interactions, and responsive design.

---

## 🚀 Technologies Used
- ⚛️ React.js  
- 🎨 Tailwind CSS & DaisyUI  
- 🧠 JavaScript (ES6+)  
- 🔔 React-Toastify  
- 📦 JSON (Product Data)

---

## ✨ Features
- 🛒 Add to cart & remove from cart functionality  
- 🔔 Toast notifications for user actions  
- 📱 Fully responsive and clean UI design
